## 计算机计数单位

| 缩写      | 英文         | 计量单位<br />（单位字节） | 计量单位<br />（单位字节） | 10进制近似<br />（单位字节） | 举例                                |
| --------- | ------------ | -------------------------- | -------------------------- | ---------------------------- | ----------------------------------- |
| B         | Byte         |                            | 2^0^                       | 10^0^                        |                                     |
| KB        | Kilobyte     | 千                         | 2^10^（2^10^=10^3^=1024）  | 10^3^                        | 16位系统能使用64K大小随机访问存储器 |
| MB        | Megabyte     | 兆                         | 2^20^（即：2^10^*2^10^）   | 10^6^                        |                                     |
| GB（KMB） | Gigabyte     | 吉                         | 2^30^                      | 10^9^                        | 32位系统能使用4GB大小随机访问存储器 |
| TB        | Trillionbyte | 太                         | 2^40^                      | 10^12^                       | 现硬盘空间最常用单位                |
| PB        | Petabyte     | 拍                         | 2^50^                      | 10^15^                       | （0.1光年）                         |
| EB        | Exabyte      | 艾                         | 2^60^                      | 10^18^                       | 64位能使用16EB大小随机访问存储器    |
| ZB        | Zettabyte    | 泽                         | 2^70^                      | 10^21^                       |                                     |
| YB        | Yottabyte    | 尧                         | 2^80^                      | 10^24^                       |                                     |
| BB        | Brontobyte   |                            | 2^90^                      | 10^27^                       |                                     |

注意：（一般不能脱离“位”谈位和字节，n位字节可以表示的地址空间数目*字节后，依然是字节的形式，但一般只能转换一次）

- 位转个单位，$n位=2^n$
- 字节转个单位，$n字节位=8n位=2^{8n}$
- 计算机单位之间不能直接相乘，必须先把其中一个转换成“个”，才能相乘
- 建议把表示地址的都加上“地址”这两个字
- 记忆口诀：千兆吉太、拍艾泽尧
