# 速录系

## E迅速录

参考：

- https://www.bilibili.com/video/BV1wf4y1h7Gp

特点

- 左右手各12个键、正交键盘，一侧上中拇各4、4、2个键
- 包括声码、韵码、功能码在内的12键位码
- 主要声码功能码在中间，韵码和部分声码在两侧

### 24键介绍 

![image-20220710111037419](速录系.assets/image-20220710111037419.png)

各个键位

| 键位 | 手指   | 读音                     |

| G    | 食指   | Ge                       |
| D    | 食指   | De                       |
| Z    | 食指   | Zhi，非~~Zi~~            |
| W    | 食指   | Wa，单独使用是删除和省略 |
| I    | 中指   | I，yi                    |
| U    | 中指   | U，wu                    |
| N    | 无名指 | En                       |
| E    | 无名指 | E和Ei                    |
| A    | 尾指   | A                        |
| O    | 尾指   | O (哦)、UO (我)          |
