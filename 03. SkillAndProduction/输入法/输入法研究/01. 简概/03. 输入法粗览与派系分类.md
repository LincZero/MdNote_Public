# 输入法粗览与派系分类

## 分类

### 按音形分类

- 纯音
- 纯形
- 音+形

### 按上屏码长分类

- 不自动上屏
- 四码自动上屏
- 三码自动上屏
- 空码即上屏，码长任意

### 按广泛度分类

设计输入法一般有两种目的，一种是为了普及，一种是是为了追求极限。
当然这没有优劣之分，只是目的不同

1. 大多数情况下，都处于两者之间。像上限码这种完全不考虑人的因素的除外
2. 即便是号称难度很高的乱序类输入法，每天练习下，入门时间也不会太长。入门后的练习也只是水磨功夫

综上，不予分类

### 按码元或选重键数分类

按码元

- 低码元：手机的9键输入法
- 中码元：手机的14、16键输入法
- 25码元：五笔系
- 26码元：拼音或大多数输入法
- \>26码元：矧码等

按选重键

- 无重自顶
- 1~3重
- 6~9重
- 选重键作为码元

### 形码 - 按字根分区分类

- 五笔系：根据键盘方位分横竖撇点折区，而且一般为25码元的方案
- 郑码系：根据A~Z字母顺序分横竖撇点折区
- 二笔系：和五笔一样有横竖撇点折区，但多了第二笔的子分类
- 仓颉系：根据A~Z字母顺序分几个 “含义区”
- 乱序类：字根乱序，且一般字根较多、有大字根

### 形码 - 按取根方式分类

单字取法

- 特征码
- 首末：小鹤音形
- 虎码（双编码）
  - 单字
    - 字根字：**Aa**
    - 二根字：**ABb**
    - 三根字：**ABCc**
    - 四根字：**ABCD**
    - 大于四根：**ABCZ**
  - 词
    - 二字词：取两个单字的前两码
    - 三字词：前2个字的首码 + 末字的前2码
    - 四字词：四个字的 首码
    - 五及以上词：前3个字的首码 + 末字的首码
- 星空键道6
  - 单字
    - 声母+空格 (一简字) / 声韵+空格 (二简字) / 声韵+笔*n
  - 词
    - （2字词不在首候选，可依次追加该词中每个字的首笔形码，直到该词出现在首候选）
    - 二字词：声韵声韵 / 声韵声韵+笔*n / 声声+空格 (简码词)
    - 三字词：声声声 / 声声声+笔*n
    - 四字词：声声声声/  声声声声+笔*n
- 五笔系
