### 染色和扩散问题

点按弱弱风，长按弱风

E顺序

- 怪身上无元素而自身火元素：起跳 -> 风伤+染色 -> 落地 -> 染色弱火伤害 -> 风伤+扩散火（成功减火抗和加火伤）
- 怪身上火元素
  - 长按：起跳 -> 风伤+扩散火+染色火 ->  落地 -> 染色弱火伤害 -> 风伤+可能扩散火（成功减火抗和加火伤）
- 怪身上水雷元素
  - 起跳 -> 风伤+扩散水雷+染色水（成功减水雷抗加水雷伤） ->  ...
- 怪身上冻水元素
  - 起跳 -> 风伤+扩散水冰+染色水（成功减水冰抗加水冰伤） ->  ...
- 怪身上水元素且自身火元素
  - 长按：起跳 -> 风伤+**扩散水+染色火** ->  落地 -> 染色弱火伤害 -> 风伤+扩散火（成功减水火抗和加水火伤）
- 怪常驻水元素且自身火元素：起跳 -> 风伤+扩散水+染色火 ->  落地 -> 染色弱火伤害 -> 风伤+只能扩散水（成功减水抗和加水伤）

Q

- 拔剑瞬间就判定完染色了，圈内所有元素按火水雷冰选出最高优先级

Q顺序

- 风伤+扩散 -> 染色伤害 (附加元素伤害)
  这样做的好处：~~可以尽量地让怪附着染色元素~~。可以染火扩水，染火扩雷
  比如：群怪此时身上是弱水，染色会上弱火
  - 先扩散后染色：将弱水扩为强水、上火变弱水、循环
  - 先染色后扩散：上火变弱弱水，弱风直接把水给浇灭了
