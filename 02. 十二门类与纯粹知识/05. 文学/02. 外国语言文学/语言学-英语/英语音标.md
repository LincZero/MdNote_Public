# 英语最重要的音标体系

音标体系

- **DJ音标**（广义国际音标，不是严格意义上的国际音标）名字由来其发明者琼斯教授（*Daniel Jones*）广泛被国内词典使用（牛津英汉，剑桥，朗文等）
    - `IPA63`，1963年的老版本
    - `IPA88`，1988年的新版本（`/i/`>`/ɪ/`，`/ə:/`>`/З:/`，`/ɔ/`>`/ɒ/  `）
- **KK音标**（美国英语）在台湾必去被广泛使用
- **韦氏音标**，韦氏词典里的音标（美国人查的词典）
- **Respelling**，用常用词里的音去标注，如`Diarrhoea`为`dye-uh-ree-uh`

补充

- **国际音标**（严格意义上）19世纪一群语言学家发明出来的为了给世界上尽可能多的不同语言注音的音标
- **DJ音标和国际音标渊源**：DJ音标的发明借了国际音标IPA的很多注音符号
- **标注对象**：英式英语和美式英语都可以标

# 音标分类

- 元音（20）
    - 单元音（12）（根据舌位分）
        - 前元音（4）`/i:  ɪ  e  æ/`
        - 中元音（3）`/З:  ə  ʌ/`
        - 后元音（5）`/ɑː  ɔː  ɒ  uː  ʊ/`
    - 双元音（8）
    - `/eɪ  aɪ  ɔɪ  aʊ  əʊ  ɪə  eə  ʊə/`
    
- 辅音（28）
    - 爆破音 / 塞音（6）
    - 摩擦音（10）
    - 破擦音（6）
    - 鼻音（3）
    - 舌边音（1）
    - 半元音（2）

## `同位异音`或`音位变体`（*allophone*）

- 比如p

    - 如果后有元音，发送气音（*aspirated*），“婆” 的声母

    - 如果后紧跟s，不送气音（*unaspirated*），不是 “菠” 的声母

    - 如果后紧跟某些辅音，嘴巴闭上就基本完成发言（基本听不出），无声除阻音（*unreleased stop*），也称“不完全爆破音”

        如：trapdoor, help me, leap year

- 比如t

    - 如果后有元音，发送气音（*aspirated*），“兔” 的声母

    - 如果后紧跟s，不送气音（*unaspirated*），不是 “刀” 的声母

    - 如果后紧跟某些辅音，嘴巴闭上就基本完成发言（基本听不出），无声除阻音（*unreleased stop*），也称“不完全爆破音”

        如：setback, forget me

    - 美式还有闪音（*flap*）

- 比如d

- 比如k

    - 如果后有元音，发送气音（*aspirated*），“开” 的声母

    - 如果后紧跟s，不送气音（*unaspirated*），不是 “该” 的声母，如sky

    - 如果后紧跟某些辅音，嘴巴闭上就基本完成发言（基本听不出），无声除阻音（*unreleased stop*），也称“不完全爆破音”

        如：trapdoor, help me, leap year

# 英美发音区别

bath，ask，glass，在英式英语发/ɑː/，美式英语发/æ/

hot，top，box，在英式英语发/ɒ/，美式英语发/ɑː/

# 疑问

- parameter中，me音标为/mI/，但有道词典发/mae/音
- 音标组合问题
    - exception的/iksepfn/和except/iksept/
    - Instrumental的/mentl/和普通的/ment/
- 其他补充
    - /jʊə/和/ʊə/不是同一个音
- /k/不见了
    - fixed的/k/音不见了，只剩下/st/
    - access的/ˈækses/ 的/k/音不见了
- fragment的/'frægmənt/和/flag的/flæg/的读法区别
- access的/ækses/和assets的/æsets/的区别
- horizon和horizontal的前面的读音完全不同
    - 重读音节变了，在前一个单词里，i处于重读音节，所以字母发本身音，第二个单词里，i处于非重读音节，所以发了相对的短音
    - 没太懂
