## 大陆优化

如果您在中国大陆，可从顶部状态栏中切换网络状态，即可屏蔽掉无法访问的网站

如果您身居海外或位于港澳等第，又或是您所在公司正式申请并得到电信部国际联网许可，可放心使用

**【免责声明】本站不协助或提供任何vpn等翻墙工具，若有特殊需求，请勿打扰！**

## 适配模式

若您希望显示的内容能更多些，点击右侧菜单栏![image-20200614082136809](04.%20网站模式.assets/image-20200614082136809.png)按钮，以自动填满

又或者您的显示器较小，无法显示所有内容，也可通过该适配模式进行解决

## 网页块说明

标签说明

- 左上：网站所需联网情况：即中国大陆是否能够访问

- 右上：网站标签

- 左下：网站语言。语言表：

| 语言简写 | 名称                                         |
| -------- | -------------------------------------------- |
| zh       | 中文（默认不显示）                           |
| en       | 英语                                         |
| mul      | 多重（网站内可切换多重语言，但必须包括中文） |
| jp       | 日语                                         |
| kor      | 韩语                                         |
| fra      | 法语                                         |
| ru       | 俄语                                         |

- 右下：点击量。无论你是点击进入，还是在搜索栏中快捷进入，都会增加点击量
