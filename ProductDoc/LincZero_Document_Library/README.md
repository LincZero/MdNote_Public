# LincZero_Document_Library

如果你在网络博客上看到这篇文章，那么对应的网络博客就是这个项目啦

技术：

- 主要框架：VuePress-Hope-Theme
- 在此基础上，使用VuePress的Alias别名系统替换了多个组件，也自己写了一些相关的插件
- 实现本地编辑与博客预览体验的一致性：
  - 将Obsidian的一些插件，如 AnyBlock，重写为Markdown-it插件的版本，以在VuePress上使用
  - 将VuePress的一些插件，如 Markdown-it-container 语法，重写为Obsidian插件的版本，以在Obsidian上使用
  - 修改了样式、侧边栏组件逻辑等
