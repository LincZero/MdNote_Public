# QtCreator

## 增加效率的快捷键

高频使用

- `Ctrl+鼠标`/`F4`/`F2`，头文件和源文件之间切换
- `F1`，帮助文档
- `Ctrl+/`，快速注释

中度使用

| 快捷键           | 作用                 |
| ---------------- | -------------------- |
| Ctrl + Shift + T | 查看类的基类和派生类 |

几乎不用

| 快捷键           | 作用                         |
| ---------------- | ---------------------------- |
| Ctrl + Shift + I | 查看包含该类的所有文件       |
| Ctrl + Shift + U | 查看Qt源代码中使用该类的文件 |

## 善用树形列表

常用

- 项目
- 大纲
- 类型层次
