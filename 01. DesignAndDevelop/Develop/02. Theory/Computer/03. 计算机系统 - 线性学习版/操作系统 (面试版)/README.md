# 操作系统 (面试版)

参考：[老汤将底层基础 —— 操作系统](https://www.bilibili.com/video/BV1U24y157BW)，讲得不错，进度：合集18/25

## 功能

**操作系统避免了应用程序与硬件交互** (网卡、内存、磁盘、显卡)。操作系统中间层，下与硬件交互，上提供系统调用

- 对上，给应用程序提供服务
  - 控制、调度程序
  - 提供服务
- 对下，对硬件资源进行管理。
  - 硬件资源协调，不同程序访问同一硬件资源

## 组成

- 内核
  - cpu管理
  - 内存管理
  - 磁盘管理
  - 中断处理 (IO设备驱动)
- 外核 (linux shell、windows 界面)
