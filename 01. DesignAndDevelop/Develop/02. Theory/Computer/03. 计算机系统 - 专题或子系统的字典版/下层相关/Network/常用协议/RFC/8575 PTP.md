# PTP（精确时间协议，Precision Timing Protocol）







# 非对称网络的时钟偏移估

## 参考文献

纯算法：

- title: Clock Offset Estimation for Systems With Asymmetric Packet Delays

  author: Ha Y.; Pak E.; Park J.; Kim T.; Yoon J.W.

  brief: 以线性微分方程(LDE)的形式定义不对称，并利用LDE来定义和排除不对称诱发的误差。无需专门硬件的成本

  links: https://www.lunlunapp.com/newsDetails/ad7beb7503600d325b7ebbad316c7232,
  https://www.techrxiv.org/articles/preprint/Clock_Offset_Estimation_for_Systems_with_Asymmetric_Packet_Delays/18393800

硬件类：

- title: WDM网络中PTP协议的一种非对称时延算法

  author: 孙捷, 曹睿, 邵晓璇

  link: https://www.fx361.cc/page/2017/0109/21341206.shtml

  brief: 在WDM网络中采用2个不同波长λ1、λ2和3步测试来实现的

- title: 传输网中PTP非对称性自动补偿的一种方法 

  author: 谢德达

  brief: 在PTP传输路径两端各增加一个光开关,从时钟获得了t1,t2,t3,t4后,切换两端开关以交换收发通道,从时钟可获得第二组t1’,t2’,t3’,t4’

  link: http://www.cqvip.com/qk/80606b/201409/663159273.html

- title: 一种PTN网络中IEEE 1588v2非对称时延的解决方案

  author: 彭良福 李有生 黄勤珍

  brief: 采用粗波分复用技术实现主从端的精确时间同步,解决了非对称时延问题。分析了色散、折射率和温度等因素对单纤双向传输时延的影响

  link: http://www.cqvip.com/qk/92171x/2013007/46636816.html

- title: 铁路时间同步网延迟不对称补偿方案

  author: 张友鹏 曹建荣 兰丽

  brief: 一种基于改进实时计算数据包排队延迟（ERTCQD）算法的补偿方案。该方案以传输链路时延为依据，得到铁路时间同步网时钟模型

  link: https://www.researching.cn/ArticlePdf/m00002/2021/58/7/0706003.pdf
