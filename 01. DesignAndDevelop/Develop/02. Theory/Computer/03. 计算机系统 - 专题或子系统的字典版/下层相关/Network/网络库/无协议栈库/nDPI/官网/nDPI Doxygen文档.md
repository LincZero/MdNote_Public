## What is nDPI

### What is nDPI

nDPI是一个DPI(深度包检测)工具包，能够:

- 检测流量上的应用协议
- 通过ET(加密流量分析)分析加密流量
- 从流量中提取选定的协议元数据
- 实现用于分析流量的api

### 版本和功能

nDPI 开发生命周期通常为 6 到 9 个月。每个版本所实现的更改和功能的历史记录都可以在其更改日志中找到。

### 安装

nDPI 是开源的，可在  [GitHub](https://github.com/ntop/nDPI) 上获取。此外，预编译的二进制 nDPI 软件包可用于 Linux 和其他平台。下面提供了二进制包的安装说明。

#### 在 Linux 上安装

安装说明可在 http://packages.ntop.org/ 找到。可以进行开发和稳定构建。稳定版本适用于生产环境，而开发版本适用于测试或早期功能访问。

#### 软件更新

有关更新软件的一般说明以及安装说明，请访问 http://packages.ntop.org/。根据操作系统的不同，nDPI 还支持通过 GUI 自动更新，如以下部分所述。

### 在 Linux 上更新软件

有关通过命令行更新软件的说明，请访问 http://packages.ntop.org/。例如，在 Ubuntu/Debian 系统上，以下命令将更新存储库、检查更新并安装最新的软件更新（如果有）：

```bash
apt-get update
apt-get upgrade
```
