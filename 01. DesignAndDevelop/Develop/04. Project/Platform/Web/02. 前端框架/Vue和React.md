## Vue和React的区别

两者作为合格前端都要学习，React对于原生JS要求会高一级
国外React用的多 国内Vue会更多

共同点

- 组件化
- 声明式编程  函数式编程（Vue3 CompositionAPI, Vue2 声明式/OptionsAPI）
- 单向数据流，数据驱动视图
- `Virtual DOM` + `Diff算法`操作DOM
- 社区成熟，都支持SSR

不同点

- Vue使用模板拥抱 html，React使用JSX拥抱JS
- Vue “自动挡”，React “手动挡” (api和生态)
