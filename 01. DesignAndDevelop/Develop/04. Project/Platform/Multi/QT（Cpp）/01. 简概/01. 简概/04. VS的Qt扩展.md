# VS的Qt扩展

## 使用方法

菜单扩展 > 管理扩展 > 搜索“Qt” > 使用扩展 “Qt Visual Studio Tools”

配置：

- 扩展 > QT VS Tools > Option > Qt >
  - General
  - Versions
    - 该配置中可以使用QT中安装好了的构建器，选择构建器的路径即可，如`QT安装路径\QT\5.15.2\msvc2019_64`
  - Legacy Project Format
