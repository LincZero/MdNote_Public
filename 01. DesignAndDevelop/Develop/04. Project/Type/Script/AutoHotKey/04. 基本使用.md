# 基本使用

## 变量类型

三种

- 字符串（文本）
- 数字（整型和浮点数）
- 对象

### 字符串

 在 AutoHotkey 中按惯例, 第一个字符在**位置 1**

### 数字

AutoHotkey 支持这些数字格式:

- 十进制整数, 如 `123`, `00123` 或 `-1`.
- 十六进制整数, 如 `0x7B`, `0x007B` 或 `-0x1`.
- 十进制浮点数, 如 `3.14159`.

### 对象

对象协议

- **Get(获取)** 一个值.
- **Set(设置)** 一个值, 使用 `:=`.
- **Call(调用)** 一个方法, 使用 `()`.



### ~~布尔~~

- AutoHotkey 没有特定的布尔类型, 因此它使用整数值 `0` 表示假, 而 `1` 表示真.
- 判断真假时, 空白或零值被认为是假, 而所有其他值被认为是真

### ~~空值~~

- AutoHotkey 没有特定的布尔类型, 一个空字符串(长度为零的字符串) 通常具有这个含义
- 如果一个[变量](https://wyagd001.github.io/zh-cn/docs/Concepts.htm#variables)或参数被称为 "空" 或 "空白", 通常意味着一个空字符串
