# 对资源的检索和下载

## 资源检索意识

首先应该培养一种意识：当你遇到一个问题的时候，要善于是查资料，而不是想不起来去查

例如有的人遇到问题就自己琢磨问题出在哪，或者一查就能查到的东西却非要问群友……



如何培养检索意识？
降低查找门槛和提高检索资源的质量是关键，下面的方法的最终目的其实都是提高检索效率和检索资源的质量，或检索性价比的方法。

1.   培养习惯，多强迫自己去检索，降低性价比阈值。

     有时你可能在路边看到一棵比较漂亮的树，你想知道这是什么树可以用手机拍照后识图，但你往往会嫌麻烦而选择放弃。
     因为你认为哪怕知道这棵树是什么名字，对自己的帮助并不大，搜索性价比低。

     强迫自己去检索不是让你事无大小都是查，如果是的话或许人就一天到晚泡在维基里了。
     而是稍微降低性价比阈值，减少自己的惰性。至于这个性价比的阈值是多少，最终还是要交给自己去把控。

2.   随着科技的提高，我们的检索会越来越方便。

     例如chatgpt就很好，如果能在群里放个chatgpt的机器人也是极好的。
     它使得你能避免搜索引擎的大量广告，以及在多个网页中寻找优质答案。

     未来随着bing速度的提高，以及语音识别和响应速度的提高，问chatgpt就像是对话一样流畅。
     未来如果还有脑机接口和脑芯片，甚至可以做到思考即检索

3.   学习检索技巧、和收集检索器资源，知道怎么检索最为有效。

     -   检索技巧：

         例如百度或谷歌引擎的高级搜索技巧、本地搜索文件或文件内容的技巧、通过通配符和正则进行搜索，等等

     -   检索器资源：

         在检索特定类别的资源时，直接使用搜索引擎可能并不高效。

         例如格式类型资源：视频学习者会直接在B站抖音上搜索学习视频、看书籍的人会在pdf检索网站上搜索，
         再例如领域类的资源：科研者在论文网站、设计者在素材网站、编程者在编程博客或论坛上 等，这可能会比直接使用搜索引擎更高效
         再例如经验心得一点的东西：问群友或去论坛问，是比较有用

     -   高级检索：（选用）

         指的是爬虫或数据分析这种
