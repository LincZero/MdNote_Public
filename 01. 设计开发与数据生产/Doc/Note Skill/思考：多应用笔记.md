## 对比各个软件

双方优缺点

|                | Typora | Obsidian |
| -------------- | ------ | -------- |
| 书写体验       | 更好   |          |
| 代码、公式体验 | 更好   |          |
| 插件体验       |        | 更好     |
| 管理体验       |        | 更好     |



## 选用

我目前是Typora和Obsidian一起用
什么笔记在哪里写？

- Obsidian
  - 日记（依赖日记插件）
  - 对话记录（依赖Chat View QQ插件）
- Typora
  - 代码、公式  
- QQ
  - 云端类



## 插件开发原则

依次遵守（越往后越不能违背，靠前的可以轻微违背）
1. 优先不搞特殊语法
2. 没有插件时，仍能较好使用
   即特殊语法依然能够较好地正常编辑和正常查看，不会严重影响笔记者的使用体验
3. 搞特殊语法不能与md语法相冲突
