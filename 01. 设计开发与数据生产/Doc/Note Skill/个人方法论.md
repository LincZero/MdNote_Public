## 学习书

> #### 根据详细程度

| 阅读方法    | 适用场景                                                 |
| ----------- | -------------------------------------------------------- |
| 逐字阅读    | 后复习阶段（高三）、简单内容的自学（初一高一书）         |
| 目的性阅读* | 学前预习、快速了解、不需要学太深                         |
| 全例题阅读  | 少例题的书的学习阶段，如高中课本                         |
| 略则读      | 跟课后                                                   |
| 课前预习    | 特点：没懂的部分不一定就要当场完全弄懂                   |
| 应试目的    | 特点：例题必须看，课后题若时间不允许还可以略看或侧重性看 |

新笔记/读书方法:查略
能查询的而非了解性质的章节，或对现时参考性较低的章节，记住其位置后略过
