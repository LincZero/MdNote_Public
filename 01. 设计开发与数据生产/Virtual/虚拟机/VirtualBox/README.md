# README

`VirtualBox`是一款跨平台的、开源的虚拟机软件。

`VirtualBox`是由德国`Innotek`公司开发，由`Sun`公司出品的软件，                        使用`Qt`编写，                        在`Sun`被`Oracle`收购后正式更名成`Oracle VM VirtualBox`。`Innotek`以`GPL`释出`VirtualBox`，并提供二进制版本及OSE版本的代码。

`VirtualBox`可以安装`Solaris`、`BSD`、`GNU/Linux`、`Windows`、`DOS`、`OS/2Warp`等操作系统。

`VirtualBox`官网：[https://www.virtualbox.org](https://www.virtualbox.org/)，或者去他父公司那下载：https://www.oracle.com/jp/virtualization/technologies/vm/downloads/virtualbox-downloads.html

## 7.x 注意项

我从 6.x 进行升级后，安装时报错：`invalid installation directory`

解决方法：

一句话总结就是：7.0.16只能安装到系统盘下，从virtualbox7.0.14到7.0.16版本更改旨在防止[将 VirtualBox](https://www.virtualbox.org/wiki/VirtualBox) 安装到普通用户能够写入或重命名内容（文件/目录）的目录中

要么装系统盘要么选择降低版本
