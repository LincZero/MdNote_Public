# 【快捷键】1-视图与变换

## 视图与显示操作

| 快捷键     | 视图与显示操作 | 补充 |
| ---------- | -------------- | ---- |
| CA+Q       | 最大化视图     |      |
| ~          | 切换视图       |      |
| 0          | 相机视图       |      |
| 拖拽左下角 | 自定义视图     |      |
| A+Z        | 透明显示       |      |

## 变换操作

| 快捷键            | 变换操作     | 补充        |
| ----------------- | ------------ | ----------- |
| T                 | 变换         | [Translate] |
| G，GXYZ，G+Sh+XYZ | 移动         | [Move]      |
| R，RXYZ，R+Sh+XYZ | 旋转         | [Rotate]    |
| S，SXYZ，S+Sh+XYZ | 缩放         | [Scale]     |
| B，Sh加选，Ct减选 | 框选         | [BOX]       |
| W                 | 切换选择模式 | [Winodw]    |
| A                 | 全选         | [All]       |
| AA                | 全不选       | [All]       |
| Ct+I              | 反选         | [Invert]    |
| Alt+GRS           | 重置变换     |             |
| Ctrl+A            | 冻结变换     |             |
