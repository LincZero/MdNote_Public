# Worm/LNK.Ippedo

也叫：Worm:W32/Ippedo，Worm是蠕虫的意思。

- 原资料：https://www.f-secure.com/v-descs/worm-w32-ippedo.shtml
- 卡饭讨论：https://bbs.kafan.cn/thread-2094612-1-1.html

U盘多了一个隐藏文件夹Skypee，感染后在每个文件夹生成文件夹名.lnk文件

> 检测 Worm:W32/Ippedo 可识别 Ippedo 蠕虫使用的恶意快捷方式 (.LNK)  文件来引诱用户无意中启动其恶意代码。一旦在计算机上活跃，远程攻击者就可以引导该蠕虫执行各种恶意操作，包括执行广告点击、将其他文件下载到计算机上、从中窃取信息、重新启动或关闭系统等。

我笔记本电脑中过这个。初步怀疑是当时19年在部队时，被同学的U盘给感染了。这玩意挺可怕的一点是能通过U盘传播
