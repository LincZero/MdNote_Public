# ComfyUI实战_真机

## 真机环境 (Windows11)

之前都是在虚拟机那边折腾，本来也是想多用虚拟机的……

太麻烦了，还是用真机吧……（最主要的难点是显卡直通）

Windows有两种方式

- 预构建
  - [官方有打包](https://docs.comfy.org/get_started/pre_package)
  - 秋叶大佬也有打包
- 手动安装

### 预构建方式

由于秋叶大佬有段时间没更新了，我用回官方的方式，使用预构建方式

Github下载也比秋叶分享的那个夸克网盘快多了，夸克还卡99.99%进度。如果网络不允许就用秋叶的吧

若为 `NVIDIA GPU` 使用 `run_nvidia_gpu` 运行它

==**这个预构建方式好快！！！牛逼**==

双击的瞬间就打开了 `http://127.0.0.1:8188/`，完全不需要再下载任何环境、编译任何东西
