# 欢迎来到 Vue Flow！

Vue Flow 是通往交互式流程图和图形世界的桥梁，使您能够为流程图和图形表示带来活力和交互性。无论是制作个人图表、生成动态编辑器，还是您想象到的任何其他内容，Vue Flow 都是您的创意伴侣。

Vue Flow 允许集成您自己的定制节点和边缘，从而轻松定制和扩展基本功能。背景、小地图和控件等附加组件进一步丰富了界面，将您的创作转变为引人入胜的平台。

如果您渴望立即开始并掌握一些代码，请探索示例或查看 CodeSandbox 模板！

## 主要特点 ​

### 无缝设置 ​

Vue Flow 让您快速开始行动。凭借元素拖动、缩放和平移以及选择等内置功能，Vue Flow 开箱即用。

### 可定制 ​

Vue Flow 由您来塑造。从自定义节点和边到连接线，您可以扩展 Vue Flow 的功能以满足您的创意需求。

### 高效、反应灵敏 ​

Vue Flow 会反应性地跟踪更改，确保仅重新渲染必要的元素。

### 实用程序和可组合性 ​

Vue Flow 专为复杂用途而设计，具有内置图形帮助器和状态可组合函数。

### 附加组件 ​

Vue Flow 附带了补充组件来增强用户界面。

- [背景](https://vueflow.dev/guide/components/background.html)：Vue Flow 提供两种内置模式，以及高度、宽度或颜色等进一步的配置，以实现个性化。
- [小地图](https://vueflow.dev/guide/components/minimap.html)：此功能可在位于右下角的小地图中提供节点的鸟瞰图。
- [控制](https://vueflow.dev/guide/components/controls.html)：Vue Flow 允许您从左下角的控制面板处理缩放功能。
- [节点工具栏](https://vueflow.dev/guide/components/node-toolbar.html)：直接从节点本身访问基本工具，以便轻松操作和控制节点
- [节点调整器](https://vueflow.dev/guide/components/node-resizer.html)：无缝调整节点的大小以满足您的需求和偏好。

### 可靠的 ​

Vue Flow 完全用 TypeScript 编写，确保为开发人员提供可靠且安全的体验。
