# Colab

## 介绍

- 官网：https://colab.research.google.com/（需要翻墙）
- 教程：官网自带基本教程
- 作用：基本同jupyter，可以让你在浏览器中编写和执行Python代码，并且：
  - 无需任何配置
  - 免费使用 GPU（见下 “与Jupter区别”）
  - 轻松共享（所以你能很轻松部署Stable Diffusion）



与Jupyter区别

- 最显著的区别的于一个在本地部署，一个是使用 Google 的云服务器

  与Jupyter相比，免费的很慢，但可以通过付费提升性能，从而达到可以使用云运算处理深度学习的地步

  但真的贵，Pro版10美元/月，Pro+版50美元/月

- 轻松分享

  但其实jupyter的ipynb文件分享也是相对方便的







- 交互式表格
- 执行历史
- 命令面板
